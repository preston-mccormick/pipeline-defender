# ⚡ Pipeline Defender ⚡

A fast-paced arcade-style defense game where you play as a technician protecting a pipeline from rust monsters and water droplets using a high-tech zapper cane.

![Pipeline Defender Screenshot](docs/screenshots/screenshot-1.png)

## Game Description

Pipeline Defender is an exciting tower defense-style game where you must protect a critical pipeline from enemy projectiles. As a skilled technician armed with a lightning zapper, you'll face increasingly challenging waves of rust monsters and water droplets that threaten to damage your pipeline.

### Key Features:
- **Dynamic Power-ups**: Collect batteries, bullhorns, survey managers, CartoPac healing items, and rare AI Bombs
- **Progressive Difficulty**: Each level increases speed and spawn rates
- **Scoring System**: Earn points for destroying enemies and collecting power-ups
- **Visual Effects**: Screen shake, explosions, lightning bolts, and dramatic power-up sequences
- **Multiple Control Options**: Play with keyboard or mouse
- **Company Branding**: Features American Innovations logo and company power-up sequences

## Controls

### Keyboard Controls:
- **Arrow Keys** or **A/D** - Move technician left and right
- **Spacebar** or **Z** - Fire the zapper
- **P** - Pause/Unpause the game

### Mouse Controls:
- **Mouse Movement** - Move technician position
- **Left Click** - Fire the zapper

### Game Controls:
- **Pause Button** - Pause the game
- **Restart Button** - Restart the current game
- **Quit Button** - Exit the game

## Power-ups

### Battery Pack (Green)
- **Effect**: Doubles zapper range for 10 seconds
- **Visual**: Cane glows yellow when active
- **Spawn**: Every 19-26 seconds
- **Points**: +250

### Bullhorn (Blue Megaphone)
- **Effect**: Creates a large blast clearing projectiles in a 200px radius
- **Visual**: Expanding orange blast ring
- **Spawn**: Every 45-68 seconds
- **Points**: +250

### Survey Manager (Clock Icon)
- **Effect**: Duplicates technician for 10 seconds
- **Visual**: Two technicians side-by-side
- **Spawn**: Once per level
- **Points**: +250

### CartoPac (Blue Circle with 'i')
- **Effect**: Heals pipeline by 10%
- **Visual**: Pipeline health increases
- **Spawn**: Every 34-45 seconds
- **Points**: +250

### AI Bomb (Rare - AI Logo)
- **Effect**: Clears ALL projectiles and shows company logo sequence
- **Visual**: Dramatic wave effect with slow-motion sequence
- **Spawn**: Every 90-120 seconds
- **Points**: +250

## Getting Started

1. **Open the game**: Simply open `index.html` in any modern web browser
2. **No installation required**: The game runs entirely in the browser
3. **Audio**: Make sure your browser allows audio for the best experience

## Game Mechanics

- **Health System**: Pipeline starts at 100% health and takes damage from enemy projectiles
- **Level Progression**: Each level lasts 30 seconds with increasing difficulty
- **Scoring**: 
  - Water droplets: 50 points
  - Rust monsters: 100 points
  - Power-ups: 250 points each
- **Game Over**: When pipeline health reaches 0%

## Technical Details

- **Built with**: HTML5 Canvas and JavaScript
- **Audio**: Web Audio API for sound effects
- **Graphics**: Custom-drawn sprites and effects
- **Browser Support**: Modern browsers with Canvas and Audio support
